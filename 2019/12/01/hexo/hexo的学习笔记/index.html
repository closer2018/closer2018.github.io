hexo上传部署命令

发表于 2019-12-01 更新于 2020-02-12 分类于 hexo 阅读次数： 阅读次数： Disqus：
本文字数： 206 阅读时长 ≈ 1 分钟

hexo上传部署三连杀命令(先cd进入git目录)

hexo clean && hexo g && hexo d

我们运行测试时，经常用到的有三个命令

hexo clean #用来清理缓存文件
hexo g      #生成文件
hexo  d   #上传到服务器

本地检测时运行

hexo s

然后打开浏览器，输入localhost:4000，就能看到博客了。
这样说明本地环境搭建完成。
