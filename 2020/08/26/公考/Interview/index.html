面试

本文最后更新于：2020年12月6日 下午

一： 主要题型及答题思路

1.1 主要题型

面视分为四大题型：

  1. 社会现象+态度观点
  2. 组织管理
  3. 应急应变
  4. 人际关系+其他题型(模拟)

1.2 评分标准

  1. 表达流畅、有气场
  2. 逻辑清晰
    1. 分点作答
    2. 重点前置
    3. 注重逻辑词连接
  3. 内容充实

1.3 答题思路

  1. 综合分析题(怎么看，要接天气)
    1. 高深新(要有作答亮点)
  2. 应急应变(怎么办，要接地气)
    1. 要有具体措施(实用、针对性强、可行度高)
  3. 组织管理和人际关系(怎么说，要接人气)
    1. 要沟通好，工作要细致、全面

二： 社会现象+态度观点

2.1 综合分析之社会现象

  1. 开头(可略过)
    1. 直接表达式
    2. 情感式
    3. 联想式
    4. 背景式
  2. 分析问题
    1. 原因： 分为主观和客观
    2. 意义
    3. 问题
  3. 解决问题(不可略过)
    1. 对策
  4. 结尾(可略过)
    1. 概括上文
    2. 点明主旨
    3. 展望未来

2.2 目的行为分析法

主体分析法分为三个主体：1.发起方、制作方 2.受众 3.政府部门、监管部门

  1. 适用范围： 政府行为、领导行为、出台政策
  2. 目的： 相对来说，说不好比较容易展开，只说好比较空洞(具体好不好还要根据实际题目判断)。出发点是好的，但是…
  3. 行为： 考虑合法性、合理性、可操作性、完善性、成本、可持续性…有哪些改善的地方
  4. 结果： 治标不治本
  5. 对策： 政策完善制定，从根本上解决问题

2.3 综合分析之态度观点

  1. 是什么
    1. 直接给出观点，然后归纳总结
  2. 为什么
    1. 论证： a.主题论证 b.正反论证 c.举例论证 d.名人名言
  3. 怎么办
    1. 启示
  4. 答题要求： 观点鲜明、论证丰富

2.4 寓言故事

需要发散思维

  1. 观点1+解释+论证+启示做法
  2. 观点2+解释+论证+启示做法
  3. 观点3+解释+论证+启示做法
  4. 启示

方法有1. 切片法 2. 原因法 3. 代入法 4. 联想法

三： 组织管理

答题方向： 对象、人员、方式、内容、效果

题型分为：

  1. 常规调研、宣传、培训、会议、比赛、考察、接待、要素轴
  2. 活动策划->时间轴
  3. 特殊类型
    1. 前期准备做什么
    2. 活动的重点难点是什么
    3. 如何保证活动的有效性

答题重点：

  1. 联系实际
  2. 着重做事
  3. 情景感、画面感

3.1 调研

  1. 前期准备： 基本信息，活动经验，物资准备
  2. 对象： 群众、政府、企业家、专家…
  3. 调研人员： 性格、专业、经验
  4. 调研方式
    1. 问卷调查
    2. 上门访谈、电话访谈
    3. 明察暗访
    4. 座谈会
    5. 文献资料
  5. 调研内容： 根据题干
  6. 调研报告
    1. 内容： 分类，对比，建议
    2. 形式： 图表，立体

3.2 宣传

  1. 前期准备： ①物资、手册、条幅 ②了解宣传内容 ③借鉴经验
  2. 宣传对象(要有针对性)： 普遍对象 + 重点对象
  3. 宣传方式：
    1. 单向宣传
      1. 传统媒体： 广播、电视、报纸、展板、宣传册、短信
      2. 新媒体： QQ、微信、公示屏、LED广告
    2. 互动宣传： ①演讲 ②征文 ③讲座 ④培训
    3. 创意宣传： ①保证书 ②责任状 ③马拉松
  4. 宣传内容： 要有针对性
  5. 宣传效果(长效性)：
    1. 反馈： ①回访 ②考核 ③问卷调查
    2. 长效机制： ①大数据 ②二次宣传

3.3 培训

  1. 对象： 对象不愿意时，管理纪律(动员参加、日常管理)
  2. 老师： 专业、经验
  3. 方式
    1. 线上和线下：
      1. 在线做题、看信息、征文
      2. 演讲、公开课
    2. 理论和实践
      1. 宣传单、材料
      2. 动手去做，如义工，注重实践
    3. 分批次和立体： 分批次培训，看效果调整培训方案
  4. 内容： 看题干
  5. 效果
    1. 反馈： 考核、回访、问卷调查、做题
    2. 长效性： ①大数据 ②二次宣传

3.4 比赛

  1. 评分标准
  2. 评委选择： 专业、权威、民主
  3. 宣传报名
    1. 宣传形式： ①公告栏 ②微信平台 ③QQ群等
    2. 报名方式： ①自荐 ②他荐 ③评选等其他报名方式
  4. 评选过程
    1. 场地
    2. 环节： ①初赛 ②复赛 ③决赛
    3. 方式： ①问卷 ②演讲 ③直播 ④面试
  5. 结果
    1. 奖励
    2. 交流会、总结征文

3.5 接待

  1. 联系确认： 了解工作内容、行程日期、人员数量和性别
  2. 准备工作： 准备工作材料、文件、成果
  3. 接站食宿： 在规定预算时间内定酒店、安排用车、餐饮
  4. 全程配合： 根据考察的内容，配合工作，提供相关文件、材料
  5. 交流反馈： 召开座谈会，交流反馈整改
  6. 安排返程

3.6 会议

  1. 准备： 根据议题收集资料和文件备用，准备场所、茶水
  2. 对象： QQ、微信、电话、短信进行通知
  3. 签到
  4. 全程
    1. 发放资料/观看视频
    2. 领导讲话(提纲准备)
    3. 工作人员/业务代表发言
    4. 交流反馈、补充意见
  5. 共享会议资料、成果或总结文档

四： 应急应变

审题(三审)： 审身份、审场景、审矛盾

类型：

  1. 一般突发事件
  2. 特殊场景突发事件(多个主体，如有群众、媒体等)
  3. 公共突发事件

答题思路： 解决当下、惩前毖后

矛盾数量

  1. 单一矛盾： 直接层层假设、先易后难
  2. 多个矛盾： 先急后缓、先重后轻、先公后私

应对媒体： ①我有错 ②不知道是否属实 ③我没错

4.1 澄清误解

  1. 主动汇报、听从指示
  2. 多方调查、搜集证据(详细展开，人证、物证)
  3. 做好解释，澄清误会(发布会、新闻稿、宣传部、网警删除)
  4. 反思总结、惩前毖后

4.2 投诉、举报、曝光

  1. 回应表态
    1. 明身份
    2. 表示重视
    3. 承诺解决
    4. 随后答复
  2. 调查取证：
    1. 对象： 当事人、目击者、监控
    2. 方式： 调研、座谈会、访谈
    3. 内容
  3. 解决问题： 分真、假情况，详细说如何处理
  4. 惩前毖后

4.3 人际关系

就事论事、一件一件说

先急后缓、先重后轻、先公后私

4.4 示例

媒体发现社会性问题如何处理？

  1. 调研取证，发现问题严肃处理；
  2. 全面排查，消除已有存在的隐患
  3. 专项治理，进行整治。加强教育、群众监督举报；
  4. 信息公开，安抚群众，表达决心；
  5. 总结上报，完善法律，厘清职责，确保长效机制。

五： 人际关系

  1. 工作为重
    1. 沟通、做事并重
    2. 生活、工作分开
    3. 犯错要先补救
  2. 适度阳光
    1. 阳光心态
    2. 认同肯定
    3. 理解包容
    4. “打小报告”
  3. 责权意识
    1. 本职工作，他人工作
    2. 不越级
    3. 领导命令
    4. 领导批评表扬
    5. 领导间冲突
  4. 原则灵活： 保持原则，讲究灵活

答题思路： 抓住矛盾点、摆平所有人、搞定所有事

六： 其他题型

6.1 情景模拟

  1. 开头： 结合身份入情入景的打开与沟通对象的“话匣子”
  2. 模拟内容： 人际关系和应急应变
  3. 结尾： 自然结尾，突显说服效果

说服原则： 动之以情 晓之以理 责之以法 诱之以利 疏堵结合
