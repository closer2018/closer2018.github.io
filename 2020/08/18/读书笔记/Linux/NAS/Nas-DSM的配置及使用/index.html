Nas_DSM的配置及使用

本文最后更新于：2021年1月21日 晚上

本文记录群晖安装完后的一些基本设置。

一： 获取root权限

  1. NAS_DSM控制面板 -> ‘终端机和SNMP’ ，开启Telnet和SSH功能。
  2. 用ssh软件登录NAS
  3. 输入sudo -i获取权限和设置root密码
  4. 设置root账号密码，输入synouser --setpw root 你的root密码这里的密码最好和admin密码一样，这样不容易搞错。
  5. vi /etc/ssh/sshd_config，将#PermitRootLogin prohibit-password改为PermitRootLogin yes
  6. service sshd restart或者/etc/initd.d/sshd restart，重启sshd服务。

二： 安装transmission

  1. 套件中心 -> 设置 -> 套件来源 -> 新增 -> http://packages.synocommunity.com -> 确定
  2. 套件中心 -> 社群 -> 安装transmission -> 设置默认下载位置/volume1/download(按需设置) -> 设置账号密码
  3. 安装好看的皮肤。用ssh软件登录NAS。
  4. wget -N https://github.com/ronggang/transmission-web-control/raw/master/release/install-tr-control-cn.sh --no-check-certificate
  5. 国内镜像wget -N https://gitee.com/culturist/transmission-web-control/raw/master/release/install-tr-control-gitee.sh
  6. sudo bash install-tr-control-gitee.sh，然后选择1。(如果提示没有权限Permission denied，则赋予权限chmod +x install-tr-control-cn.sh。UI界面升级也是进入该目录，执行此命令)
  7. 共享文件夹 -> 编辑 你的共享文件夹 -> 权限 -> 本地群组 -> sc-download 可读写
  8. transmission里面设置远程端口或frp添加服务列表
  9. 开启NAS防火墙相关端口

三： 安装kodi组建家庭影院

首先安装kodi：

  1. kodi官网下载安装包，建议下载17.6版本，然后升级高版本。
  2. 电视上安装apk文件
  3. 进入app，设置里面，interface settings -> skin -> font设置为Arial based
  4. interface settings -> regional -> language设置为简中

接下来设置群晖共享SMB：

  1. 控制面板文件服务 -> 开启SMB
  2. 新建用户账号kodi -> 设置密码 -> 用户组users ->共享文件夹只读 -> 默认下一步到应用

登录kodi，在资料库添加windows网络 SMB，选择NAS，添加视频源。

设置群晖共享NFS：

  1. 控制面板文件服务 -> 开启NFS
  2. 共享文件夹 -> 选中编辑 -> NFS权限 -> 新增 -> ip为* -> 权限只读 -> Squash设置为映射root为admin -> 允许来自非特权端口的连接 -> 允许用户访问已装载的子文件夹 -> 确定应用

登录kodi，在资料库添加NFS，选择NAS的ip，添加视频源。

字幕插件推荐shooter、opensubtitles.org、subscenne.com三个
