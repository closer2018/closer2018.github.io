1.1 开启SSH访问

1.3 修改时间

使用date 语句查看时间，如不正确，则：首先在终端中用下面的命令安装ntpdate工具

sudo apt install ntpdate

启用NTP

sudo timedatectl set-ntp true

继续使用date语句查看时间，如不正确，则：

sudo dpkg-reconfigure tzdata

修改时区为Asia Shanghai，继续使用date语句查看时间并确认正确
