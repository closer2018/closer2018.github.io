零： 搜索引擎之搜索技巧

  1. 排除关键词(减号+关键词)
  2. 精确搜索(给关键词加引号)
  3. 指定网站内搜索(site:域名 关键词)
  4. 指定文件格式(filetype:文件格式 关键词)
  5. 指定标题搜索(intitle:关键词)
  6. 指定范围搜索-搜索结果正文包含关键词(intext:关键词 allintext:关键词1 关键词2… 关键词n)
  7. 指定范围inurl-搜索结果的url中包含关键词字符串(inurl:url关键词)
  8. 叠加使用

注意事项：
1.所有的冒号都是半角，也就是英文的冒号，而不是中文的冒号
2.空格很重要，关键词之间要有空格
3.提取关键词很重要(可以提高搜索效率)

一： 如何使用搜索栏

所有搜索需求用空格隔开。

搜索栏输入：

  1. 搜索项目名里面包含python、项目readme里面包含python、描述里面包含python
    in:name python、in:readme python、in:description 爬虫/in:des 爬虫
    PS：description描述里面包含比较准确

  2. stars数大于1000、forks数大于xxx
    stars:>1000、forks:>xxx

  3. 指定编程语言
    language:python

  4. 最后更新时间限定
    pushed:>2020-01-01

  5. 限定国内开发者
    location:China

二： chrome插件推荐

  • Octotree
  • GitZip for github
    双击你想下载的文件后面空白的地方，文件前方会出现√，右下角出现Download checked items即可下载。
  • 英文不好的小伙伴推荐沙拉查词

三： git clone速度慢的解决方案

登录Gitee -> 新建仓库 -> 拉到最下的创建上面，点击导入已有仓库 -> 填入你想git clone的GitHub仓库地址 -> 本地，git clone。
