判断推理

本文最后更新于：2020年12月8日 上午

判断推理：主要测查应试人员对各种事物关系的分析推理能力，涉及对图形、语词概念、事物关系和文字材料的理解、比较、组合、演绎和归纳等。常见的题型有：图形推理、定义判断、类比推理、逻辑判断等。S

一： 图形推理

1.1 位置规律

识别： 元素组成相同

  1. 平移
    1. 方向： 上下、左右、顺逆时针、外圈、行列
    2. 步数： 恒定、递增(等差)
  2. 旋转
    1. 方向： 顺逆时针
    2. 角度： 45°、90°、180°
  3. 翻转
    1. 方向： 上下(横轴对称)、左右(竖轴对称)
  4. 技巧
    1. 比较思维： 既可以比较题干，也可比较选项
    2. 九宫格，中间图形特殊优先“米”字型观察

1.2 样式规律

识别： 元素组成相似(相同 图形或线条 重复出现)

  1. 遍历(缺啥补啥)
    1. 元素遍历
      1. 每个元素出现次数相同
      2. 特定元素的遍历
    2. 位置遍历
      1. 外框的遍历
      2. 内部线条/阴影/空白/图形的遍历
      3. 涉及多个相同小元素时，考虑数量遍历
  2. 加减同异
    1. 相加、相减
    2. 求异
    3. 求同
    4. PS: 位置和样式的复合考法
  3. 黑白运算
    1. 特征： 图形轮廓和分隔区域相同，内部颜色不同
    2. 方法： 相同位置运算
    3. 小技巧： 优先考虑黑白运算后，从问号处着手解题更快，边找边验证
    4. 特别注意： “白+黑”和“黑+白”是不同等式

1.3 属性规律

识别： 元素组成不同

  1. 对称性(优先)
    1. 轴对称：对称轴方向、数量
    2. 中心对称
  2. 开闭性： 优先全开放、全封闭
  3. 曲直性： 优先全曲或全直，其次曲+直
  4. 凹凸性
    1. 任意两点相连，在图外，是凹，如×
    2. 任意两点相连，在图内，是凸，如O

1.4 数量规律

识别： 元素组成不同且无属性规律，数量规律明显

  1. 点
    1. 特征
      1. 线条交叉明显
      2. 乱糟糟一团线交叉
      3. 相切较多
    2. 细化考法
      1. 交点(切点)
      2. 端点
      3. 直曲相交
  2. 线
    1. 直线数特征图
      1. 多边形、单一直线
    2. 曲线数特征图
      1. 曲线图形(全曲线图、圆、弧)
    3. 笔画数
      1. 一笔画图形
        1. 连通图
        2. 奇点数=0或2
      2. 多笔画图形
        1. 笔画数=奇点数/2
        2. 如果存在多部分，每个部分的笔画单独算，再相加
      3. 常见特征图
        1. 五角星、圆和圆相切、田和日的变形
  3. 角
    1. 个数
    2. 角度(钝角，锐角)
    3. 数＜180°的角，角不重复数
    4. 特征图： 扇形、折线图、改造图
  4. 面
    1. 个数
    2. 面积
    3. 特征
      1. 图形被分割，封闭面明显
      2. 生活化图形，粗线条图形中留空白区域
  5. 素
    1. 元素种类
    2. 元素个数
      1. 满眼相同图形，重复数
      2. 三角、四边形
      3. 曲直
    3. 部份数
      1. 线条连在一起叫一部分
      2. 常考查粗线条图形

1.5 空间重构

1.5.1 四面体

  1. 基础知识
    1. 展开图中构成一条直线的两条边，折合之后是同一条边
    2. 平行四边形展开图中两条短边，折合之后也是同一条边
  2. 解题方法-箭头法
    1. 对题干和选项中同一个面，画箭头
    2. 箭头方向是否一致

1.5.2 六面体

  1. 相对面
    1. 两个面在同一行或同一列，且中间隔了一个面
    2. 两个面在“Z”字形两端，且紧靠着“Z”字形中间那条线
    3. 相对面只能出现一个，且必须出现一个。
  2. 相邻面
    1. 在展开图中，呈直角的两条边，折合后为同一条边
    2. 看图形相对位置
    3. 画边法：选特殊面的唯一点或唯一边，顺时针(或逆时针)画边，与题干不匹配的排除。

1.5.3 八面体

  1. 基础知识
    1. 展开图中，构成“V”字形的两条边，折合之后是同一条边
    2. 每个顶点发出4个面
    3. 每个面只有3个相邻面
  2. 解题方法
    1. 箭头法
    2. 公共点法

1.5.4 图形平面拼合

  1. 平行等长消除法(找到相对面积较大的图形作为基准图形，找到平行且等长的边进行拼合，拼合之后等长线条抵消)
  2. 特殊边秒杀法
    1. 找出四个图形中不可能消除的线条(特殊边)，如没有平行的线条，最长的线条，唯一的横直线
    2. 到选项中去找这条线，没有的直接排除，剩下的用平行等长消除法拼合
  3. 注意比例坑，江苏题经常在此出坑。

1.5.5 截面图

  1. 解题原则： 一刀切
  2. 截面有弧线
    1. 立体图形本身存在曲面(如圆柱、圆锥等)、拦腰切、斜切
  3. 截面有矩形
    1. 六面体-- 拦腰切，上下切，斜切
    2. 圆柱-- 上下切
  4. 截面有三角形
    1. 六面体-- 从棱上某一个点开始切
    2. 圆锥-- 经过顶点向底面切
  5. 小技巧
    1. 六面体切不出直角三角形
    2. 立体图形有曲线，才能切出带曲线的截面
    3. 圆柱切不出梯形

二： 定义判断

  1. 识别五要素(问属于)
    1. 主体
    2. 客体
    3. 出发点(造成的结果)
    4. 过程
    5. 落脚点(场地)
  2. 问不属于
    1. 直接看选项，找出不同的一项，同构选项排除法

三： 类比推理

3.1 逻辑关系

  1. 全同关系
    1. 中外翻译 eg罗曼蒂克:浪漫
    2. 古称和今称 eg北平:北京
    3. 谦称和敬称 eg犬子:令郎
    4. 俗称和雅称 eg三月:桃月
    5. 全称和缩写 eg辽宁:辽
  2. 包含关系
    1. 一重包含、二重包含
      1. 一重包含 eg苹果:水果
      2. 二重包含 eg红富士:水果
    2. 种属包含、部分包含
      1. 种属 eg苹果:水果；杂志:期刊
      2. 整体和部分 eg阳光:紫外线；树木:森林
      3. 不可分割 eg鱼:鱼鳔
      4. 功能互补 eg键盘:电脑；羽毛球:羽毛球拍
  3. 并列关系
    1. 方法： 构成并列关系的概念必须是同一层次上的，如国画:油画都是画、香蕉:苹果都是水果
    2. 矛盾： 非此即彼 eg生死:物质意识 黑夜白天：男人女人
    3. 反对： 除了A和B以外，还有其他情况 eg红色:黄色
  4. 交叉关系
    1. 方法： 用有的……是……造句来判断
    2. eg 男人:老师；唐诗:七律；瓷器:茶杯
  5. 全异关系
    1. 含义： 词语所表示的概念属于同一大类的事物，或具有某种共同性。所有A不是B，且A和B之间不是并列关系
    2. eg实数:木耳；蝙蝠:鸟类；天鹅:塑料
  6. 无对应关系
    1. 两个词语之间没有一点联系。看起来特别匪夷所思的两组词，有可能适用这个考点。
    2. eg布告:棉布

3.2 语义关系

  1. 近义关系
    1. 同义词或近义词 eg安居乐业:丰衣足食、谨慎:慎重
    2. 属性相近 eg寒冷:冰冷；吼叫:咆哮
  2. 反义关系
    1. eg节约:浪费；早餐:傍晚
  3. 褒贬关系
    1. eg后果:结果；妄想:遐想:理想，贬:中:褒
  4. 象征关系
    1. eg水:温柔；土:敦厚

3.3 描述关系

  1. 与事物相关
    1. 事物与其性质 eg花:香
    2. 事物与其象征意义 eg凤凰:吉祥
    3. 事物与其活动空间、制作场所、原材料、所在地、作用对象等等 eg飞机:天空、大夏:城市、桌子:木材
    4. 同样功能的古代、现代物品 eg蒲扇:冷空调、火柴:打火机
  2. 与人相关
    1. 职业与工作地点
      1. eg编辑:杂志社、农民:土地
    2. 职业与其特征、用具等其他相关内容
      1. eg士兵:军装、学生:桃李

3.4 条件关系

  1. 因果关系
    1. 根本原因(本质原因) eg疾病:抑郁，因为疾病所有抑郁
    2. 直接原因(导火索) eg努力:成功，因为努力而成功。
    3. 间接原因 eg招摇撞骗:锒铛入狱，因为招摇撞骗所有锒铛入狱。
  2. 时间顺成关系
    1. 几个动作或事件相继发生，有一定的时间先后顺序。B在A之后发生。
    2. eg起跑:冲线、起床:刷牙、校对:印刷:出版
  3. 目的关系
    1. 某个事件的发生是以另一个事件为目的
    2. A是B的目的，B是A的手段。eg广告:宣传、节食:减肥
  4. 必要充分条件
    1. 充分条件
    2. 必要条件
      1. eg消毒:手术；启动:驾驶

3.5 语法关系

  1. 词性： 名词、动词、形容词
  2. 主谓关系(谓语指干了什么，一般是动词+形容词) eg轮船:启航；宝石:发光
  3. 动宾结构 eg讨论:问题；合同:定制；风卷:残云
  4. 偏正结构
    1. 一个词对另一个词起修饰或限制作用。
    2. eg清秀:面容；热烈:欢迎；义务:劳动；窗户:玻璃，玻璃做的窗户；戒指:宝石，宝石做的戒指。
  5. 并列结构
    1. 两个词性相同的实词并列组成
    2. eg七上:八下；天南:地北
    3. 成语题目 eg围湖造田:饮鸠止渴=忍辱偷生:削足适履
  6. 后补结构 eg安于现状:轻于鸿毛:对答如流
  7. 排列类比(江苏特色) eg:dda:22v
    1. 方法：先找出唯一不同的元素，排除选项后再用相同的找。

3.6 常识知识

  1. 生活常识
    1. 功能对应 eg药物:治疗；滑板:运动
    2. 组成对应 eg骨骼:人体、梁柱:房屋
    3. 加工对应
      1. 物理变化 eg木材:家具、花岗岩:雕塑
      2. 化学变化 eg葡萄:酒；大豆:酱油
    4. 职业对应
    5. 空间对应 eg小鸟:天空:飞翔；蟋蟀:草丛:欢叫
    6. 属性对应
      1. 必然属性 eg糖:甜
      2. 或然属性 eg官员:廉洁；子女:孝顺
    7. 民俗对应
      1. eg七夕:织女；端午:屈原
  2. 书本常识
    1. 地理 eg布达拉宫:拉萨；云冈石窟:大同
    2. 文学历史
    3. 科普
      1. eg时间:速度；体积:密度
      2. eg三角形:三棱柱；四边形:正方体

3.7 二级辨析

解题技巧:

  1. 遣词造句法
  2. 对比词性
  3. 横纵对比法
  4. 感情色彩是否一致
  5. 范围程度是否有差异
  6. 性质、功能是否相同
  7. 词的构成方式是否相同
  8. 必然与或然
  9. 全面与片面
  10. 绝对反义和相对反义

四： 逻辑推理

逻辑推理题目错误选项常见类型：

  1. 主体不一致
  2. 话题不一致
  3. 明文规定、道听途说的选项不选
  4. 不明确选项。如“目前不知道”
  5. 强弱有别
  6. 时间重合≠因果关系，一定错，不选。

4.1 翻译推理

识别： 题干中出现可推出/不可推出

  1. 前推后
    1. 关键词： 如果……，那么……；若……，则……；只要……，就……；所以……，都……；为了……一定……；……是……的充分条件；等
    2. 翻译规则： 前推后：A → B
  2. 后推前
    1. 关键词： 只有……才……；不……不……；除非……否则不……；……是……的基础/假设/前提/关键/必要条件
    2. 翻译规则： 后推前： B → A
    3. 谁必要谁放箭头后面
    4. PS： 除非(只要/其他……)A否则B 等价于 -B → A;
  3. 逆否等价: A → B 等价于 -B → -A
  4. 且和或
    1. 含义
      1. A或B： 二者至少一个成立为真
      2. A且B： 二者同时成立为真
    2. 推理规则： 或命题为真时， 否1 → 1
  5. 德摩根定律
    1. -(A且B) = -A 或 -B
    2. -(A或B) = -A 且 -B
    3. 小技巧： 逐项添符号，且或互变
  6. 推理方法： 推理结构保持完全相同

4.2 组合排列

识别： 题干给出两组及两组以上对象，问给出几组对象之间的关系

  1. 排除法、代入法
    1. 题干信息确定优先使用排除法
    2. 题干信息不确定优先使用代入法
  2. 辅助技巧
    1. 最大信息： 以题干中出现次数最多的信息为推理起点
    2. 符号(＞、＜)
    3. 表格(一维表、二维表)

4.3 削弱题型

  1. 识别方法
    1. 设问中包含“反驳、削弱、质疑”的论证题目
    2. 问不能削弱/质疑分2种情况
      1. 无关选项
      2. 加强选项
  2. 削弱方式
    1. 削弱论点(最常考，力度最强)
      1. 通过矛盾命题或例子的形式直接削弱题干观点
    2. 削弱论证(力度第二)
      1. 拆桥： 能够证明论据无法直接得出论点
    3. 削弱论据(力度最弱)： 证明论据时错的

4.4 加强题型

  1. 识别： 设问中包含“支持、加强、前提/假设”条件
  2. 加强方式
    1. 搭桥(最强)
      1. 题干特征
        1. 论点和论据出现不同概念
        2. 论点和论据为整体和部分关系
      2. 选项特征： 同时包含论点和论据中的关键词，并肯定论点和论据之间的关系
    2. 必要条件(第二)
      1. 题干特征
        1. 提问为前提、假设、必要条件、加强论证、无搭桥项时考虑必要条件
        2. 如：题干是对比试验，但起点是否一致未知
        3. 如：论点是比例，论据只涉及其中一项，需要补充分母
      2. 选项判别： 否定代入。先假设选项是错误的，代入题干论点不成立，则可以证明该选项是必要条件。
    3. 补充论据
      1. 解释原因(第三)： 说明论点成立的原因
      2. 举例论证(第四)： 说明论点成立的例子
  3. 加强力度： 搭桥＞必要条件＞解释原因＞举例论证

4.5 日常结论

日常结论不找对的，用排除法，找错的三个排除。

识别： 题干为类言语的表述，无逻辑关联词，提问方式为“可以推出”，“不能退出”……

  1. 三不选
    1. 逻辑错误不选
    2. 无中生有不选
    3. 偷换概念不选
  2. 四慎选
    1. 绝对化的词： 一定、肯定、必须、所有、如果……就……、只有……才……
    2. 比较性的词： 更、比、越来越……
    3. 程度性的词： 最、主要、次要、首要
    4. 范围扩大缩小。
  3. 一优选
    1. 在选项都存在瑕疵时，带有表述“可能”意思字眼的选项优选。

4.6 真假推理

题干特征： 题干中出现几个命题，但其真假未知，要求通过判断命题的真假情况推出某些结论

  1. 优先找关系
    1. 矛盾关系
      1. 特性： 必然一真一假
      2. 使用方法
        1. 首先找矛盾，绕过矛盾看其余
        2. 条件一假，假话在矛盾中，其余全真
        3. 条件一真，真话在矛盾中，其余全假
      3. 常见矛盾关系
        1. A与-A
        2. “所有A都是B”与“有的A不是B”
        3. “所有A都不是B”与“有的A是B”
        4. “A→B”与“A且-B”
        5. “A或B”与“-A且-B”
        6. “A且B”与“-A或-B”
    2. 反对关系
      1. 使用方法
        1. 没有矛盾找反对，绕过反对看其余
        2. 条件一真，找到“有的”和“有的不”，其余全假
        3. 条件一假，找到“所有都”和“所有都不”，其余全真
      2. 常见反对关系
        1. “有的A是B”与“有的A不是B”： 必有一真
        2. “所有A都是B”与“所有A都不是B”： 必有一假
    3. 推出关系
      1. 如果命题P为真，那么命题Q也一定为真，即P(真)→Q(真)，P和Q就是推出关系
      2. 使用方法
        1. 找不到矛盾或反对，找推出关系
        2. 一真前假，一假后真
      3. 常见推出关系
        1. A(真) → A或B(真)
        2. 所有A都是B(真) → 某个A是B(真) → 有的A是B(真)
        3. 所有A都不是B(真) → 某个A不是B(真) → 有的A不是B(真)
  2. 找不到关系： 考虑代入排除法

五： 小技巧

  1. 图形推理一般每道题会考一个知识点，做题的时候把相应的知识点标记在题目旁边，其他题目的考点大概率不会重复，所以做其他题目的时候可以不考虑那个知识点，提高做题速度。
  2. 图推题目没有思路，一般考虑下一笔画，还是没有思路超过30秒放弃，不纠结。
  3. 类比推理善用造句。
  4. 定义判断就是找茬，找主体、客体、定语(定语是用来修饰、限定、说明名词或代词的品质与特征的。)、状语(一种状态)的不同。
