3.2 进入docker容器及拷贝

一：如何进入docker容器？比较简答的方法如下：

  1. 首先docker ps查看容器id；
  2. 假设容器id前5位是faf2b，执行docker exec -it faf2b /bin/bash，就进入了容器内部。

二：如何向容器内拷贝文件？

  1. 首先docker ps查看容器id；
  2. 假设容器id前5位是faf2b，执行docker cp /root/拷贝的文件名 faf2b:/home/其余路径
  3. docker exec -it faf2b /bin/bash，进入容器查看文件是否拷贝成功。
